# Gap analysis

## Présentation du gap analysis
Le principe repose sur la comparaison entre la situation actuelle et la situation projetée, puis identifier les leviers et tâches pour supprimer cet écart. Cette méthode est utile pour la rédaction de la [note de cadrage](../assets/note_de_cadrage.docx), livrable finale de la phase de [cadrage](index.md).

## Etapes d'un gap analysis 
1. Réaliser une photo de la situation présente : *Où suis-je ?*
-> Dresser un inventaire des points caractérisant l'état initial. 
Pour réaliser cette étape : Brainstorming, Interviews, ...

2. Situation projetée : *Où veux-je être ?*
Utiliser la Méthode SMART pour définir des bons objectifs (ex : *Nous voulons être les numéros 1* -> *Nous visons 30% de part de marché sous 2 ans*)

3. Identifier les leviers pour atteindre l'état souhaité : *Comment y aller ?*
- pour chaque écart : repérer les facteurs explicatifs de cette différence (peuvent être qualitatifs comme quantitatifs)
- Pour facteur, identifier les leviers actionnables qui permettent de réduire le gap = les actions à menées
