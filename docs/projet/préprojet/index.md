# Pré-projet

La phase de pré-projet consiste à mesurer la pertinence du projet avant de le lancer. 

Chaque projet représente un investissement de ressources (argent comme humain), les enjeux économiques, organisationnelles ou stratégiques doivent être évalués.

Le livrable finale de cette phase est le [business case](business_case.md) (ou étude d'opprotunité).

Retrouvez ici mon [template de business case](assets/business_case.docx).
